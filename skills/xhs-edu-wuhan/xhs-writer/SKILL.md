# xhs-writer

必须先读取：
- `clients/wuhan-tutoring/references/文案风格指南.md`
- `clients/wuhan-tutoring/prompts/system-prompt.md`

生成内容时：
- 保持武汉本地升学、家长决策、中考提分语境
- 标题优先结果导向和问题导向
- 正文只在有足够正文级样本时再做更强风格归纳

局部修改规则：
- 用户只说改标题：只改标题
- 用户点名某一段：只改该段
- 未点名的已确认内容不得修改
